## ADIM 1: Kayıt Olun (2 dakika)

```
[ ] Platforma gidin → "Ücretsiz Kayıt Ol" butonuna tıklayın
[ ] Ad Soyad, E-posta, Telefon, Ofis Adı, İlçe bilgilerinizi girin
[ ] Şifre belirleyin (en az 8 karakter)
[ ] E-postanıza gelen doğrulama linkine tıklayın
[ ] Kampanya kodunuzu girin → 3 AY PRO PLAN ÜCRETSİZ!
```

---

## ADIM 2: İlk İlanınızı Ekleyin (3 dakika)

```
[ ] Sol menüden "Portföy" sekmesine gidin
[ ] "+ Yeni İlan" butonuna tıklayın
[ ] Mülk bilgilerini doldurun:
    → İlçe, Mahalle
    → Tip (Daire/Villa/Arsa)
    → Oda sayısı, Net m², Brüt m²
    → Kat, Bina yaşı, Isınma tipi
    → Fiyat
[ ] "Kaydet" butonuna tıklayın
```

---

## ADIM 3: AI Değerleme Yapın (2 dakika)

```
[ ] Eklediğiniz ilanın detay sayfasını açın
[ ] "AI Değerleme" butonuna tıklayın
[ ] Yapay zeka 24 kriteri analiz eder (2-3 saniye)
[ ] Sonuç: Tahmini fiyat + güven aralığı + emsal karşılaştırma
```

**Doğruluk oranı: %90.65** — Sektör ortalamasının çok üzerinde!

---

## ADIM 4: PDF Rapor İndirin (1 dakika)

```
[ ] Değerleme sonuç ekranında "PDF Rapor İndir" butonuna tıklayın
[ ] Profesyonel rapor hazır: fiyat + emsal + bölge + deprem riski
[ ] Müşterinize WhatsApp veya e-posta ile gönderin
```

---

## ADIM 5: Telegram Botunuzu Kurun (2 dakika)

```
[ ] Ayarlar → Telegram Entegrasyonu → "Bot Oluştur"
[ ] Ekrandaki linke tıklayın → Telegram açılır
[ ] "/start" yazın ve gönderin
[ ] Yeşil onay = Bot aktif!
```

**Bot ile:** Müşterileriniz 7/24 ilan sorabilir, fiyat öğrenebilir, size mesaj bırakabilir.

---

## TAMAMLANDI!

Artık şunları yapabilirsiniz:

| Özellik | Nasıl |
|---------|-------|
| **Fiyat tahmini** | Portföy → İlan → "AI Değerleme" |
| **Bölge analizi** | Sol menü → "Bölge Analizi" |
| **Müşteri takibi** | Sol menü → "Müşteriler" → "+ Yeni Müşteri" |
| **Eşleştirme** | Sol menü → "Eşleştirme" (otomatik çalışır) |
| **İlan metni yazma** | İlan → "İlan Asistanı" (AI ile 3 farklı ton) |
| **Kredi hesaplama** | Araçlar → "Kredi Hesaplayıcı" (6 banka) |
| **Vitrin yayınlama** | Sol menü → "Vitrin" → "Yayınla" |
| **Deprem riski** | İlan → "Deprem Riski" sekmesi |
